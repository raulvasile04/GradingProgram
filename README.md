# Grading Programm

## Description
This is a grading program that lets students look for their grade for a specific class , or lets them see their average
grade

## Tech-Stack
- Java

## Future improvements
At the time the programm is just in it's early stages . In the future , the code structure will look much better , it will have
a solid and well done and documented structure , lots more features , more course and more students and maybe a register student 
method that lets you register yourself as a student .

## How to run
### Requirements
- Java JDK 17 or newer

### Steps 
1. Clone the repository
2. Navigate into the project directory
3. Compile the program
4. Run the program

### Usage 
- Enter yes when asked if you want to look for a students grade
- Enter one of the students first- and lastname
- Choose a course :
- 1. Maths
  2. English
  3. Average grade
- The program will display the requested grade in the terminal
